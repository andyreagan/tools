Fitness Activity Viewer

A self-contained, local-first fitness activity viewer for .fit and .gpx files. No database, no backend — just a single HTML file and your browser.

Chart.js Version — Recommended

Simpler API, easy to understand, great for standard fitness visualizations. Lightweight and well-documented.

CDN Version (~55 KB) Bundled / Offline (~830 KB)

D3.js Version — Advanced

More powerful and flexible with fine-grained control. Similar to what Strava uses for its charts.

CDN Version (~55 KB) Bundled / Offline (~900 KB)

Features

  • Interactive OpenStreetMap route visualization
  • Elevation, heart rate & pace charts
  • Chart-to-map crosshair linking
  • Metric & imperial unit toggle
  • YAML metadata support
  • Photo gallery from media/ folder
  • Metadata-only mode for gym workouts
  • Works with Strava & Garmin exports
Usage: Place the HTML file alongside your activity data (activity.fit, activity.gpx, metadata.yaml) and serve with any local web server. Bundled versions inline all JS/CSS but still load map tiles from OpenStreetMap.
